## FORMULAS

**Ton-Mile Center (Center of Gravity)**

$$x^* = \frac{\sum w_i x_i}{\sum w_i}, \quad y^* = \frac{\sum w_i y_i}{\sum w_i} \quad \text{where } w_i = V_i \times R_i$$

$$ \text{and } V_i \text{ indicates the volume at location i and } R_i \text{ indicates the transportation rate at location i}.$$



**Inventory Metrics**

| Metric | Formula |
|--------|---------|
| Average Aggregate Inventory Value | $\sum (\text{Units}_i \times \text{Cost}_i)$ |
| | |
| Average Weekly Sales | $\dfrac{\text{Annual COGS}}{\text{Weeks per Year}}$ |
| | |
| Weeks of Supply | $\dfrac{\text{Avg Aggregate Inventory Value}}{\text{Avg Weekly Sales}}$ |
| | |
| Inventory Turnover | $\dfrac{\text{Annual COGS}}{\text{Avg Aggregate Inventory Value}}$
